# E-commerce Platform

A robust, full-stack E-commerce solution built with the Spring Boot ecosystem.

## 🛡️ Key Features
- **Secure Authentication**: RBAC (Role-Based Access Control) using Spring Security.
- **Product Catalog**: Dynamic browsing with search, filtering, and pagination.
- **Shopping Cart**: Real-time cart management and item review.
- **Order Management**: Secure checkout process and order tracking.
- **Responsive UI**: Modern interface built with Thymeleaf and custom CSS.

## 🛠️ Tech Stack
- **Backend**: Java 17, Spring Boot 3.2.5, JPA/Hibernate.
- **Security**: Spring Security 6.
- **Database**: MySQL.
- **Frontend**: Thymeleaf, HTML5, CSS3.
- **Build**: Maven.

## 🚀 Getting Started

### Prerequisites
- JDK 17 or higher.
- MySQL Server.
- Maven.

### Installation & Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/ecommerce-platform.git
   ```
2. **Database Setup**:
    - Create a database named `ecommerce_db` in MySQL.
    - Update `src/main/resources/application.properties` with your MySQL username and password.
3. **Run the application**:
   ```bash
   mvn spring-boot:run
   ```
4. **Access the platform**:
    - Visit `http://localhost:8080` in your browser.

## 📁 Project Structure
- `src/main/java`: Backend logic (Controllers, Entities, Services, Repositories).
- `src/main/resources/templates`: HTML views (Thymeleaf).
- `src/main/resources/static`: Assets (CSS, Images).
```
